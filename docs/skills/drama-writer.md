# drama-writer

短剧剧本创作助手 — 支持竖屏短剧和横屏微短剧的全流程创作，覆盖项目设置、大纲、角色、分集、剧本、分镜到成本预估。

## 特性

- **双平台适配** — 竖屏短剧（1-3 分钟/集）和横屏微短剧（5-15 分钟/集）
- **8 大题材模板** — 都市逆袭、都市甜宠、都市复仇、仙侠、古装宫斗、古装权谋、悬疑、穿越重生
- **专业剧本格式** — 场景标题、台词、镜头建议、音效提示，可直接用于拍摄
- **一集一钩** — 自动确保每集结尾留有悬念钩子
- **分镜提示** — 镜头类型、构图建议、转场方式
- **成本预估** — 场景分级、特效统计、服装道具需求汇总
- **素材搜索** — WebSearch / WebFetch 搜索行业趋势和创作素材
- **自动存档** — 跨会话恢复创作进度

## 安装

```bash
# 方式一：复制到 skills 目录
cp -r skills/drama-writer ~/.claude/skills/drama-writer

# 方式二：本地开发模式
claude --skill-dir /path/to/drama-writer
```

## 使用方法

### 斜杠命令

```bash
/drama                         # 使用默认路径 ~/Desktop/drama/
/drama ~/my-project            # 使用自定义路径
```

## 创作流程

| 阶段 | 说明 | 输出文件 |
|------|------|----------|
| 项目设置 | 平台类型、题材、集数、受众 | `project.md` |
| 故事大纲 | 高概念、主线、分幕、反转节点 | `outline.md` |
| 角色设计 | 档案、关系网、台词风格、出场记忆点 | `characters.md` |
| 分集大纲 | 每集核心冲突和结尾钩子 | `outline.md`（追加） |
| 剧本创作 | 专业格式剧本 | `episodes/EPXXX-标题.md` |
| 分镜提示 | 镜头、构图、转场建议（可选） | `storyboard/EPXXX-分镜.md` |
| 成本预估 | 场景分级、特效、预算参考（可选） | `cost-estimate.md` |
| 进度跟踪 | 完成集数、伏笔状态、大纲偏离记录 | `progress.md` |

## 平台参数对比

| 平台 | 时长/集 | 集数 | 台词量/集 | 场景特点 |
|------|---------|------|-----------|----------|
| 竖屏短剧 | 1-3 分钟 | 60-100 集 | 800-1200 字 | 简洁、人物为主 |
| 横屏微短剧 | 5-15 分钟 | 20-40 集 | 2000-4000 字 | 丰富、可有大场面 |

## 题材支持

| 题材 | 推荐平台 | 核心卖点 |
|------|----------|----------|
| 都市逆袭 | 竖屏 | 爽感、打脸、逆袭 |
| 都市甜宠 | 竖屏 | 高糖、嗑 CP、心动 |
| 都市复仇 | 竖屏 | 谋略、爽感、解气 |
| 仙侠 | 竖屏/横屏 | 热血、升级、斗法 |
| 古装宫斗 | 竖屏/横屏 | 智斗、权谋、逆袭 |
| 古装权谋 | 横屏 | 格局、博弈、烧脑 |
| 悬疑 | 横屏 | 烧脑、反转、推理 |
| 穿越重生 | 竖屏 | 先知、改命、爽感 |
