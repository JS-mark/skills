# iconfont-downloader

Iconfont 图标下载助手 — 从 iconfont.cn 搜索并下载 SVG 图标。支持账号密码 / 二维码登录，搜索后展示结果让用户选择下载。

## 特性

- **多种登录方式** — 账号密码或二维码扫码登录
- **中英文搜索** — 中文搜索不理想时自动切换英文关键词
- **结果展示** — 搜索后以表格形式展示，由用户选择下载
- **批量下载** — 支持序号、范围、自然语言多种选择格式
- **搜索缓存** — 同一关键词无需重复搜索
- **自定义路径** — 支持指定图标输出目录

## 安装

```bash
# 方式一：复制到 skills 目录
cp -r skills/iconfont-downloader ~/.claude/skills/iconfont-downloader

# 方式二：本地开发模式
claude --skill-dir /path/to/iconfont-downloader
```

### 依赖

Skill 会自动检测并使用以下浏览器自动化工具（按优先级）：

1. **MCP 浏览器工具** — 如果可用，无需额外安装
2. **Playwright** — `pnpm add playwright`
3. **Puppeteer** — `pnpm add puppeteer`

## 使用方法

在对话中直接说：

- "帮我下载一个首页图标"
- "搜索 iconfont 上的 home 图标"
- "找个搜索图标下载到项目里"

Claude 会自动引导完成 **登录 → 搜索 → 选择 → 下载** 的完整流程。

### 批量下载选择格式

| 格式 | 示例 | 说明 |
|------|------|------|
| 逗号分隔 | `"1,3,5"` | 下载第 1、3、5 个 |
| 范围 | `"1-5"` | 下载第 1 到 5 个 |
| 自然语言 | `"前5个"` | 下载前 5 个 |
| 全部 | `"all"` 或 `"全部"` | 下载全部 |

## 使用示例

```
用户：帮我下载几个设置相关的图标

→ 检查登录状态（未登录则引导登录）
→ 搜索 "设置"，展示表格：
  | 序号 | 名称 | 作者 |
  |------|------|------|
  | 1 | settings | 作者A |
  | 2 | gear | 作者B |
  | ...
→ 用户："下载1和3"
→ 批量下载并保存到项目 icons 目录
```

## 注意事项

1. **登录安全** — 密码仅传递给 login 工具，不会保存到磁盘
2. **Session 有效期** — session 可能过期，过期时自动引导重新登录
3. **合理使用** — 避免短时间内频繁请求
4. **版权遵守** — 下载的图标请遵守原作者的版权声明
5. **浏览器依赖** — 首次使用 Playwright / Puppeteer 可能需要下载浏览器

## 故障排除

| 问题 | 解决方案 |
|------|----------|
| 登录失败 | 检查凭据、网络，或尝试二维码登录 |
| 搜索无结果 | 尝试英文关键词，检查登录状态 |
| 下载失败 | 检查目录写入权限和磁盘空间 |
| 浏览器工具未检测到 | 安装 `playwright` 或 `puppeteer` |
